# Authentication

All protected API endpoints require a valid JWT Bearer token. Use these endpoints to obtain and manage tokens.

## POST /auth/register

Create a new Helix Aero user account and receive a JWT.

```http
POST https://api.helix.aero/api/v1/auth/register
Content-Type: application/json
```

### Request Body

<ParamField body="name" type="string" required>
  Full name of the user (2–255 characters).
</ParamField>

<ParamField body="email" type="string" required>
  Email address. Must be unique. Used as the login identifier.
</ParamField>

<ParamField body="password" type="string" required>
  Password (minimum 8 characters).
</ParamField>

<ParamField body="password_confirmation" type="string" required>
  Must match `password`.
</ParamField>

<ParamField body="locale" type="string">
  Preferred locale: `"fr"` or `"en"`. Defaults to `"fr"`.
</ParamField>

```json
{
  "name": "Jean Dupont",
  "email": "jean.dupont@example.com",
  "password": "securepassword",
  "password_confirmation": "securepassword",
  "locale": "fr"
}
```

### Response `201 Created`

<ResponseField name="data.token" type="string">
  JWT Bearer token for subsequent requests.
</ResponseField>

<ResponseField name="data.token_type" type="string">
  Always `"bearer"`.
</ResponseField>

<ResponseField name="data.expires_in" type="integer">
  Token lifetime in seconds (604,800 = 7 days).
</ResponseField>

```json
{
  "data": {
    "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
    "token_type": "bearer",
    "expires_in": 604800
  }
}
```

### Errors

| Status | Message |
|--------|---------|
| `422` | Validation errors (email taken, password mismatch, etc.) |

## POST /auth/login

Authenticate with email and password to receive a JWT.

```http
POST https://api.helix.aero/api/v1/auth/login
Content-Type: application/json
```

**Rate limit:** 5 requests / minute

### Request Body

<ParamField body="email" type="string" required>
  Registered email address.
</ParamField>

<ParamField body="password" type="string" required>
  Account password.
</ParamField>

```json
{
  "email": "jean.dupont@example.com",
  "password": "securepassword"
}
```

### Response `200 OK`

```json
{
  "data": {
    "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
    "token_type": "bearer",
    "expires_in": 604800
  }
}
```

### Errors

| Status | Message |
|--------|---------|
| `401` | `"These credentials do not match our records."` |
| `422` | Validation errors (missing fields) |
| `429` | Rate limit exceeded |

## GET /auth/me

Return the currently authenticated user's profile.

```http
GET https://api.helix.aero/api/v1/auth/me
Authorization: Bearer <token>
```

### Response `200 OK`

<ResponseField name="data.id" type="integer">
  User ID.
</ResponseField>

<ResponseField name="data.name" type="string">
  Full name.
</ResponseField>

<ResponseField name="data.email" type="string">
  Email address.
</ResponseField>

<ResponseField name="data.locale" type="string">
  Preferred locale (`"fr"` or `"en"`).
</ResponseField>

<ResponseField name="data.email_verified_at" type="string | null">
  ISO 8601 timestamp of email verification, or `null`.
</ResponseField>

<ResponseField name="data.created_at" type="string">
  ISO 8601 timestamp of account creation.
</ResponseField>

```json
{
  "data": {
    "id": 42,
    "name": "Jean Dupont",
    "email": "jean.dupont@example.com",
    "locale": "fr",
    "email_verified_at": "2026-01-15T10:30:00.000Z",
    "created_at": "2026-01-14T09:00:00.000Z"
  }
}
```

### Errors

| Status | Message |
|--------|---------|
| `401` | Missing or expired JWT |

## POST /auth/logout

Invalidate the current JWT token. The token cannot be used after this call.

```http
POST https://api.helix.aero/api/v1/auth/logout
Authorization: Bearer <token>
```

### Response `200 OK`

```json
{
  "data": {
    "message": "Successfully logged out."
  }
}
```

### Errors

| Status | Message |
|--------|---------|
| `401` | Missing or expired JWT |
