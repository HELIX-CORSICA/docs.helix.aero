# Helix Flight Planner

Helix Flight Planner is a professional desktop application for planning aerial survey missions. Designed for photogrammetry and LiDAR operators, it provides accurate flight strip computation, 3D globe visualization, multi-format export, and quality control maps — all in a modern, cross-platform interface.

## Key Features

<CardGroup cols={2}>
  <Card title="3D Globe Visualization" icon="globe">
    Plan missions on a live 3D globe with satellite imagery, terrain, and real-time flight line overlays.
  </Card>
  <Card title="Accurate Strip Computation" icon="route">
    Flight strips calculated with Vincenty geodesy for sub-meter accuracy. GSD, overlap, and sidelap update in real time.
  </Card>
  <Card title="Multi-Format Export" icon="file-export">
    Export to KML, GPX, CSV, GeoJSON, TXT, .helix JSON, and PDF mission reports with a single click.
  </Card>
  <Card title="Quality Control Maps" icon="chart-bar">
    Coverage gap maps, GSD distribution, sidelap grids, and threshold alerts before you fly.
  </Card>
</CardGroup>

## Supported Sensors

Helix Flight Planner ships with a catalog of 80+ cameras and 14+ LiDAR sensors from leading manufacturers. You can also define custom sensors with full specification control.

**Camera planning:** focal length, sensor dimensions, pixel size, forward overlap, sidelap, and GSD at any altitude.

**LiDAR planning:** FOV, pulse repetition rate, scan rate, beam divergence, swath width, and point density.
