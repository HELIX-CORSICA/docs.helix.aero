# Configuration des capteurs

Helix Flight Planner inclut un catalogue complet de capteurs et prend en charge les definitions de capteurs personnalises. Le capteur selectionne determine directement le GSD, l'espacement des bandes, la largeur de fauchee et la densite de points de votre mission.

## Catalogue de capteurs standard

L'application inclut plus de **80 cameras** et **14 capteurs LiDAR** des principaux fabricants, synchronises depuis le serveur Helix Aero. Les capteurs se mettent a jour automatiquement lorsque vous etes connecte a Internet.

**Fabricants de cameras supportes :** Sony, Phase One, Hasselblad, DJI, MicaSense, Parrot, Leica, Trimble, et plus encore.

**Fabricants de LiDAR supportes :** Velodyne, Ouster, Livox, Riegl, Leica, YellowScan, et plus encore.

Acces au catalogue depuis le panneau **Capteur** dans la vue de planification. Utilisez la barre de recherche ou filtrez par :
- **Type :** Camera ou LiDAR
- **Fabricant**

## Specifications des cameras

Lors de la selection d'une camera, Helix Flight Planner utilise les specifications suivantes pour calculer le GSD et l'espacement des bandes :

| Specification | Description | Exemple |
|---------------|-------------|---------|
| **Focale (mm)** | Focale de l'objectif | 35 mm |
| **Largeur capteur (mm)** | Largeur physique du capteur d'image | 35,9 mm |
| **Hauteur capteur (mm)** | Hauteur physique du capteur d'image | 24 mm |
| **Taille pixel (µm)** | Taille physique de chaque pixel | 4,51 µm |
| **Largeur image (px)** | Resolution en pixels dans la largeur | 7952 px |
| **Hauteur image (px)** | Resolution en pixels dans la hauteur | 5304 px |

### Formule du GSD

```
GSD_largeur (cm) = (Altitude_m × Largeur_capteur_mm) / (Focale_mm × Largeur_image_px) × 100
GSD_hauteur (cm) = (Altitude_m × Hauteur_capteur_mm) / (Focale_mm × Hauteur_image_px) × 100
```

Le GSD se met a jour en temps reel lorsque vous modifiez l'altitude ou la selection du capteur.

### Espacement des bandes depuis le recouvrement

```
Espacement_bandes (m) = Emprise_largeur_m × (1 - Recouvrement_lateral / 100)
Intervalle_prises (m) = Emprise_hauteur_m × (1 - Recouvrement_longitudinal / 100)
```

## Specifications LiDAR

Les capteurs LiDAR sont definis par la geometrie du faisceau et le debit de donnees :

| Specification | Description | Exemple |
|---------------|-------------|---------|
| **Champ de vision (°)** | Angle de vision complet en travers de piste | 120° |
| **PRR (kHz)** | Frequence de repetition des impulsions | 600 kHz |
| **Vitesse de balayage (Hz)** | Vitesse de rotation du balayage | 100 Hz |
| **Divergence du faisceau (mrad)** | Angle de cone de chaque faisceau laser | 0,5 mrad |

### Largeur de fauchee LiDAR

```
Largeur_fauchee (m) = 2 × Altitude_m × tan(Champ_vision_deg / 2 × π / 180)
```

### Densite de points LiDAR

```
Points_par_m2 = (PRR × Vitesse_balayage) / (Vitesse_sol_ms × Largeur_fauchee_m)
```

## Impact des specifications sur la planification

La selection du capteur a un effet en cascade sur toute la mission :

```
Specs capteur + Altitude
    → GSD (cm/px pour cameras, pts/m² pour LiDAR)
    → Emprise / largeur de fauchee
    → Espacement des bandes (controle par le recouvrement lateral)
    → Nombre de bandes
    → Nombre de prises / distance totale
    → Temps de vol estime
```

Modifier un parametre du capteur ou l'altitude recalcule immediatement toutes les valeurs en aval — aucune action manuelle requise.

## Capteurs personnalises

Si votre capteur n'est pas dans le catalogue standard, vous pouvez creer un capteur personnalise.

### Creer une camera personnalisee

1. Dans le panneau **Capteur**, cliquez sur **+ Ajouter un capteur personnalise**.
2. Selectionnez **Camera** comme type de capteur.
3. Remplissez les champs requis :
   - Nom du capteur (ex. : "Hasselblad L2D-20c Custom")
   - Focale (mm)
   - Largeur et hauteur du capteur (mm)
   - Taille du pixel (µm) — ou saisissez la resolution en pixels et l'application calcule la taille du pixel
4. Cliquez sur **Enregistrer**. Le capteur personnalise est disponible immediatement et synchronise avec votre compte.

### Creer un LiDAR personnalise

1. Dans le panneau **Capteur**, cliquez sur **+ Ajouter un capteur personnalise**.
2. Selectionnez **LiDAR** comme type de capteur.
3. Remplissez les champs requis :
   - Nom du capteur
   - Champ de vision (degres, angle total)
   - Frequence de repetition des impulsions (kHz)
   - Vitesse de balayage (Hz)
   - Divergence du faisceau (mrad)
4. Cliquez sur **Enregistrer**.

### Modifier et supprimer des capteurs personnalises

Les capteurs personnalises peuvent etre modifies ou supprimes depuis le panneau **Capteur**. Les capteurs du catalogue standard (synchronises depuis le serveur) ne peuvent pas etre modifies. Si vous avez besoin d'une variante, creez un capteur personnalise base sur les specifications standard.

<Warning>
  La suppression d'un capteur personnalise n'affecte pas les missions deja creees avec ce capteur. Les missions existantes conservent les specifications du capteur telles qu'elles etaient au moment de la creation.
</Warning>
