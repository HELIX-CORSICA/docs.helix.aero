# Helix Flight Planner

Helix Flight Planner est une application desktop professionnelle pour la planification de missions de releve aerien. Concu pour les operateurs de photogrammetrie, LiDAR et pushbroom, il combine un globe 3D en direct, un calcul precis du calepinage, des modes d'altitude conscients du terrain, des cartes de controle qualite et un export multi-format dans une seule interface multiplateforme.

## Capteurs supportes

Helix Flight Planner inclut un catalogue de plus de **80 cameras**, **14 capteurs LiDAR** et une bibliotheque de profils d'aeronefs, le tout curé par Helix Aero. Vous pouvez egalement definir des capteurs et aeronefs personnalises avec un controle complet des specifications.

- **Cameras** — focale, dimensions du capteur, taille du pixel, recouvrement longitudinal, lateral et GSD a toute altitude.
- **LiDAR** — FOV, frequence d'impulsion, vitesse de balayage, divergence du faisceau, largeur de fauchee et densite de points.
- **Pushbroom** — geometrie du capteur ligne et regles d'espacement natives.

Les principaux fabricants couverts incluent Sony, Phase One, Hasselblad, DJI, MicaSense, Leica, Trimble, PhaseOne iXM, Velodyne, Ouster, Livox, Riegl, YellowScan, etc.

## Modes d'altitude supportes

| Mode | Description |
|------|-------------|
| **AMSL** | Altitude absolue au-dessus du niveau moyen de la mer. Planification classique a altitude fixe. |
| **iAMSL** | Initial AMSL — altitude interpretee par rapport au niveau de la mer sur le MNT. |
| **iAGL** | Initial AGL — altitude interpretee au-dessus du sol ; les bandes suivent le MNT pour maintenir une garde sol constante. |

La source MNT est choisie dans l'assistant de mission (SRTM 30 m, Copernicus GLO-30 ou aucune pour un terrain plat).

## Configuration requise

| Plateforme | Minimum |
|------------|---------|
| Windows    | Windows 10 (64 bits) ou superieur |
| macOS      | macOS 12 Monterey ou superieur |
| RAM        | 8 Go (16 Go recommandes) |
| GPU        | Acceleration OpenGL / ANGLE requise pour le globe 3D |
| Stockage   | 500 Mo d'espace disque disponible |
| Reseau     | Requis pour la connexion, la synchronisation et le telechargement MNT ; utilisation hors ligne apres la premiere activation |
